## Setup for developers

1. Copy `.env.template` to `.env`:
```bash
cp .env.template .env
```

2. Edit `.env` and add your Mira SDK API key:
```
API_KEY=your_actual_api_key_here
```

3. Run 

```streamlit run holidayplannerapp.py```
